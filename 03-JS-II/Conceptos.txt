



BUCLE FOR:

	EL BUCLE FOR ES UN BLOQUE DE CODIGO QUE SE UTILIZA PARA REPETIR LA EJECUCION DE LINEAS DE CODIGO HASTA QUE SE CUMPLA UNA CIERTA CONDICION.

OPERADORES LOGICOS:

	&& - EL "Y" SE UTILIZA PARA EVALUAR DOS CONDICIONES, Y SI AMBAS SON VERDADERAS RETORNA 'TRUE' (VERDADERO), Y SI UNA, O LAS DOS SON FALSAS RETORNA 'FALSE'(FALSO).

	|| - CON EL "O", SI UNA DE LAS DOS ES VERDADERA RETORNA 'TRUE', Y SI LAS DOS SON FALSAS RETORNA 'FALSE'.

	! - Y EL "NO" ES UTILIZADO PARA CONTRADECIR UNA CONDICION, DECIR QUE ALGO NO ES VERDADERO, O NO ES FALSO.
